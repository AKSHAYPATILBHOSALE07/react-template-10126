Dynamic Axial Compression (DAC)

DAC stands for “Dynamic Axial Compression” and DAC Columns are a type of column used in liquid chromatography under high medium and low pressure. DAC columns are designed to allow for efficient packing of chromatography media, which results in high-resolution separations and reduced sample analysis time.

SLURRY PACKING
UNIT

For effective column packing the slurry needs to be charged effectively, to have a proper packing using DAC. A slurry charging and packing unit thus become an essential tool in preparative chromatography. Our unit is a closed mixing unit that can easily be integrated with large-diameter DAC preparative columns.


The column packing can be performed in very few simple steps:


  • Filling vessel with buffer as recommended by manufacture/R&D team followed by media to be used
  • Gentle mixing with a pneumatic agitator provided in the unit
  • Removing fine particles after the solution is at rest if required
  • Make a uniform slurry using a buffer and charge the empty column using a pneumatic slurry transfer pump
